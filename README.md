# Phase-3-Group-Capstone

## Team: BeyondWords

### Project Overview

BeyondWords is developing a machine learning model capable of recognizing American Sign Language (ASL) alphabet gestures from images. The goal of the project is to explore image classification techniques and evaluate how effectively a model can distinguish between ASL signs. The project will progress through exploratory data analysis, preprocessing, model development, evaluation, and deployment planning.

---

## Dataset

The project uses the **ASL Alphabet Dataset** from Kaggle, which contains approximately 87,000 training images distributed across 29 classes. Twenty-six classes represent the letters A–Z, while three additional classes represent SPACE, DELETE, and NOTHING.

Key characteristics of the dataset include:

* Approximately 87,000 training images
* 29 total classes
* 200 × 200 image resolution
* Consistent background and camera angle
* Roughly balanced class distribution
* Limited subject and environmental variation

Dataset Link:
https://www.kaggle.com/datasets/grassknoted/asl-alphabet
